# Бот Глобального Гносиса для ВКонтакте

## Описание
Python-бот для ВКонтакте, использующий библиотеку vk_api с поддержкой LongPoll.

## Структура проекта
- `main.py` — основной файл бота
- `hello.info` — текст приветствия бота
- `requirements.txt` — зависимости (vk_api, python-dotenv)
- `start.sh` — скрипт запуска (устанавливает пакеты и запускает бота)

## Настройка
Для запуска бота необходимо задать переменную окружения `VK_TOKEN` — токен сообщества ВКонтакте.

## Запуск
```bash
bash start.sh
```
