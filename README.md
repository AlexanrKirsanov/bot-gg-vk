# bot-gg-vk
Бот Глобального Гносиса для ВКонтакте

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Создайте файл `.env` и укажите токен сообщества ВКонтакте:
```
VK_TOKEN=ваш_токен_сообщества
```

3. Запустите бота:
```bash
python3 main.py
```

## Описание
Бот отвечает на приветствия и сообщает пользователям о себе.
Находится в стадии разработки.
